# Batch Rename Tool
## Function
Batch rename all files in a specified folder with custom prefix and sequence number.

## Environment Requirements
- Python 3.8+
- No additional dependencies

## Running Steps
1. Clone the repository: `git clone https://github.com/nqilin/python-small-tools.git`
2. Navigate to the tool directory: `cd python-small-tools/File-Processing/Batch-Rename`
3. Run the script: `python batch_rename.py`
4. Enter folder path and prefix as prompted
